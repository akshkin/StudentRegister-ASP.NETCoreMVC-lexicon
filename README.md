# Student Register
A simple and clean ASP.NET MVC application for managing students.

## Features
- View a list of all students
- Seed data with 3 students
- Add new students
- Edit existing students
- Delete students with a confirmation modal
- Server-side validation
- Entity Framework database integration
- Details page of each student
- Search students by last name
- Sort students by first name or last name

## Project setup
- Clone the project using `git clone`
- Update the connection string in `appsettings.json` if using a different connection string
- Run `update-database` from the Package Manager Console
- Run the project and navigate to the url shown in the terminal
